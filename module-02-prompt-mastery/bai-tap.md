# 📝 Bài tập Module 2: Prompt Mastery — Thực hành viết prompt

> **Mục tiêu:** Áp dụng công thức SHARP để viết prompt ảnh/video cho sản phẩm gia dụng.
> **Thời lượng:** ~45 phút
> **Bloom Level:** Understand + Apply
> **Cần chuẩn bị:** 3 sản phẩm gia dụng đã chọn ở Module 1

---

## Bài tập 1: Viết 3 prompt ảnh sản phẩm theo công thức SHARP

### Yêu cầu

Chọn **3 sản phẩm gia dụng** (có thể dùng 3 sản phẩm đã chọn ở Module 1).

Với mỗi sản phẩm, viết **1 prompt ảnh hero shot** theo công thức SHARP:

| Yếu tố SHARP | Sản phẩm 1: ________ | Sản phẩm 2: ________ | Sản phẩm 3: ________ |
|--------------|----------------------|----------------------|----------------------|
| **S** – Scene (Bối cảnh) | | | |
| **H** – Hero (Sản phẩm) | | | |
| **A** – Action (Hành động) | | | |
| **R** – Result (Kết quả) | _có thể bỏ trống_ | _có thể bỏ trống_ | _có thể bỏ trống_ |
| **P** – Props (Đạo cụ) | _có thể bỏ trống_ | _có thể bỏ trống_ | _có thể bỏ trống_ |

Sau đó **ghép thành prompt hoàn chỉnh** và chạy trên Google Flow Imagen 4.

### 💡 Ví dụ mẫu

**Sản phẩm: Máy rửa bát mini**

| Yếu tố | Nội dung |
|---------|---------|
| **S** | Bếp nhỏ trong căn hộ chung cư, tông trắng-gỗ, gọn gàng |
| **H** | Máy rửa bát mini màu trắng, cửa kính trong suốt, đèn LED xanh bên trong |
| **A** | Bên trong máy có bát đĩa đang được rửa, tia nước phun nhẹ |
| **R** | Bát đĩa sạch bóng xếp gọn trên kệ |
| **P** | Khăn lau trắng gấp gọn, bình hoa nhỏ trên kệ |

**Prompt hoàn chỉnh:**
> Bếp nhỏ trong căn hộ chung cư phong cách tối giản, tông trắng-gỗ. Máy rửa bát mini màu trắng đặt trên mặt bếp, cửa kính trong suốt để lộ bên trong có bát đĩa đang được rửa, tia nước phun nhẹ, đèn LED xanh sáng. Bát đĩa sạch bóng xếp gọn bên cạnh. Khăn lau trắng gấp gọn và bình hoa nhỏ trên kệ. Ánh sáng tự nhiên từ cửa sổ.

### ✅ Checklist hoàn thành

- [ ] Đã viết 3 prompt theo SHARP (có thể bỏ R/P cho hero shot)
- [ ] Đã ghép thành 3 prompt hoàn chỉnh
- [ ] Đã chạy ít nhất 1 prompt trên Imagen 4
- [ ] Đã lưu kết quả ảnh

## Bài tập 2: Viết 3 prompt video Hook (3 giây đầu)

### Yêu cầu

**Hook** (3 giây đầu tiên của video, dùng để thu hút người xem dừng lại) là phần **quan trọng nhất** — quyết định người xem lướt qua hay dừng lại.

Viết **3 prompt video hook** cho **3 sản phẩm khác nhau**. Mỗi hook dùng 1 kiểu khác nhau:

| Hook # | Sản phẩm | Kiểu hook | Prompt |
|--------|---------|-----------|--------|
| 1 | ________ | **Vấn đề – Giải pháp** (show vấn đề trước) | |
| 2 | ________ | **Wow moment** (show kết quả "wow" ngay) | |
| 3 | ________ | **Câu hỏi** (tạo tò mò) | |

### 💡 Ví dụ mẫu

**Kiểu "Vấn đề – Giải pháp" — Quạt tích điện:**
> Cận cảnh khuôn mặt cô gái đang đọc sách ngoài trời nắng, mồ hôi lấm tấm trên trán, biểu cảm khó chịu. Bàn tay đặt chiếc quạt tích điện mini màu pastel trước mặt. Quạt bật, tóc bay nhẹ, cô gái mỉm cười. Ánh nắng vàng, chuyển động chậm.

**Kiểu "Wow moment" — Robot hút bụi:**
> Góc quay thấp ngang sàn nhà: robot hút bụi trắng lướt qua, phía sau để lại vệt sàn sáng bóng. Camera theo chuyển động robot, phía trước là bụi bẩn, phía sau sạch bong. Chuyển động mượt mà, ánh nắng chiếu chéo.

### ✅ Checklist hoàn thành

- [ ] Đã viết 3 prompt hook với 3 kiểu khác nhau
- [ ] Mỗi prompt đủ chi tiết (bối cảnh, sản phẩm, chuyển động, ánh sáng)
- [ ] Đã chạy ít nhất 1 video hook trên Veo 3.1

## Bài tập 3: A/B Testing — So sánh 2 version prompt

### A/B Testing là gì?

**A/B Testing** (Tạo nhiều phiên bản video để so sánh hiệu quả): tạo **2 phiên bản khác nhau** cho cùng 1 mục tiêu, xem bản nào tốt hơn. Đây là kỹ năng quan trọng để tối ưu video affiliate.

### Yêu cầu

Chọn **1 sản phẩm gia dụng**. Viết **2 prompt video khác nhau** cho cùng sản phẩm đó:

| | Version A | Version B |
|---|-----------|-----------|
| **Sản phẩm** | (giống nhau) | (giống nhau) |
| **Bối cảnh** | | |
| **Góc quay** | | |
| **Ánh sáng** | | |
| **Chuyển động** | | |
| **Prompt đầy đủ** | | |

Chạy cả 2 prompt trên Veo 3.1 và so sánh kết quả:

| Tiêu chí đánh giá | Version A (1-5 điểm) | Version B (1-5 điểm) |
|-------------------|----------------------|----------------------|
| Chất lượng hình ảnh | | |
| Sản phẩm có nổi bật không? | | |
| Có muốn xem tiếp không? (hook) | | |
| Phù hợp đăng TikTok? | | |
| **Tổng điểm** | | |

### ✅ Checklist hoàn thành

- [ ] Đã viết 2 prompt khác nhau cho cùng 1 sản phẩm
- [ ] Đã chạy cả 2 trên Veo 3.1
- [ ] Đã so sánh và chấm điểm
- [ ] Đã ghi chú: Version nào tốt hơn và **tại sao**?

## Bài tập 4: Tạo prompt template cá nhân

### Yêu cầu

Chọn **1 sản phẩm yêu thích** (sẽ dùng lại ở Module 3 trở đi). Tạo **bộ prompt template cá nhân** gồm:

1. **1 prompt ảnh hero shot** (cho thumbnail)
2. **1 prompt video hook** (3 giây đầu)
3. **1 prompt video demo** (8 giây show tính năng)

Dùng format sau:

```
📦 SẢN PHẨM: [tên sản phẩm]

🖼️ PROMPT ẢNH HERO SHOT:
[prompt ảnh]

🎬 PROMPT VIDEO HOOK (3s):
[prompt video hook]

🎬 PROMPT VIDEO DEMO (8s):
[prompt video demo]
```

💡 **Mẹo:** Lưu template này lại! Bạn sẽ dùng lại ở Module 3 khi tạo video hoàn chỉnh.

- [ ] Đã viết 1 prompt video hook
- [ ] Đã viết 1 prompt video demo

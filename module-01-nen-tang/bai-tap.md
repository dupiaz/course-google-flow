# 📝 Bài tập Module 1: Nền tảng — TikTok Affiliate & Google Flow

> **Mục tiêu:** Thực hành đăng ký, khám phá, và chọn sản phẩm.
> **Thời lượng:** ~30 phút
> **Bloom Level:** Remember + Understand

---

## Bài tập 1: Đăng ký & Khám phá Google Flow

### Yêu cầu

1. Truy cập [flow.google.com](https://flow.google.com) và đăng nhập bằng tài khoản Google
2. Chọn gói **Free** (để dùng thử)
3. Khám phá **4 tab chính**: Video, Image, Scene, Chat
4. Thử tạo **1 ảnh đơn giản** bằng Imagen 4:

   **Prompt mẫu để thử:**
   > Một chiếc máy xay sinh tố màu xanh mint đặt trên bàn gỗ, bên cạnh là trái cây tươi, ánh sáng tự nhiên từ cửa sổ

5. Chụp màn hình kết quả và lưu lại

### ✅ Checklist hoàn thành

- [ ] Đã đăng nhập Google Flow thành công
- [ ] Đã khám phá 4 tab (Video, Image, Scene, Chat)
- [ ] Đã thử tạo 1 ảnh bằng Imagen 4
- [ ] Đã chụp màn hình kết quả

## Bài tập 2: Tìm hiểu TikTok Shop Affiliate

### Yêu cầu

**Nếu bạn ĐÃ có 1.000+ followers:**
1. Mở TikTok → Trung tâm người sáng tạo → TikTok Shop → Affiliate Program
2. Đăng ký làm Affiliate Creator
3. Vào Marketplace → tìm kiếm **"gia dụng"** hoặc **"đồ nhà bếp"**
4. Chọn 1 sản phẩm bất kỳ → ghi lại: tên sản phẩm, giá, % hoa hồng

**Nếu bạn CHƯA có 1.000 followers:**
1. Mở TikTok → Trung tâm người sáng tạo
2. Ghi lại số followers hiện tại: ___________
3. Tìm hiểu mục Affiliate Program → đọc điều kiện tham gia
4. Ghi chú lại những điều kiện bạn đã đáp ứng và chưa đáp ứng

### ✅ Checklist hoàn thành

- [ ] Đã truy cập mục TikTok Shop Affiliate
- [ ] Đã đọc điều kiện tham gia
- [ ] Đã ghi lại thông tin cần thiết (followers/sản phẩm)

## Bài tập 3: Chọn 3 Winning Product gia dụng

### Winning Product là gì?

**Winning Product** (Sản phẩm có tiềm năng bán chạy) là sản phẩm có:
- 🔥 Nhu cầu cao (nhiều người cần)
- 💰 Margin tốt (hoa hồng cao)
- 📱 Phù hợp video ngắn (dễ demo, dễ gây ấn tượng)
- ⚡ Yếu tố "wow" (khiến người xem dừng lại)

### Yêu cầu

1. Vào TikTok Shop Marketplace **hoặc** lướt TikTok với hashtag `#giadung`, `#smarthome`, `#tiktokmademebyit`
2. Chọn **3 sản phẩm gia dụng** mà bạn muốn làm affiliate
3. Với mỗi sản phẩm, điền vào bảng sau:

| Tiêu chí | Sản phẩm 1 | Sản phẩm 2 | Sản phẩm 3 |
|----------|-----------|-----------|-----------|
| **Tên sản phẩm** | | | |
| **Giá bán** | | | |
| **Hoa hồng (%)** | | | |
| **Số đã bán (trên TikTok)** | | | |
| **Lý do bạn chọn** | | | |
| **Điểm "wow" để làm video** | | | |

### 💡 Gợi ý sản phẩm gia dụng hot 2026

- Robot hút bụi / máy lau sàn
- Máy xay sinh tố cầm tay
- Nồi cơm điện thông minh
- Quạt tích điện cầm tay/để bàn
- Máy lọc không khí
- Đồ decor nhà cửa đa công dụng
- Máy rửa bát mini
- Bếp từ tiết kiệm điện

### ✅ Checklist hoàn thành

- [ ] Đã chọn 3 sản phẩm gia dụng
- [ ] Đã điền đầy đủ bảng phân tích
- [ ] Đã xác định được "điểm wow" cho mỗi sản phẩm

# 📝 Bài tập Module 3: Quy trình sản xuất — Tạo 1 video affiliate hoàn chỉnh

> **Mục tiêu:** Áp dụng quy trình 6 bước để tạo 1 video affiliate hoàn chỉnh từ đầu đến cuối.
> **Thời lượng:** ~120 phút
> **Bloom Level:** Apply + Analyze
> **Cần chuẩn bị:** 1 sản phẩm gia dụng đã chọn ở [Module 1](../module-01-nen-tang/bai-tap.md) + prompt template cá nhân từ [Module 2 — Bài tập 4](../module-02-prompt-mastery/bai-tap.md)

## 🎯 Bài tập lớn: Tạo 1 video affiliate hoàn chỉnh (30 giây)

Đây là bài tập **tổng hợp** — bạn sẽ đi qua toàn bộ 6 bước đã học trong bài học Module 3.

### Sản phẩm sử dụng

Lấy **1 sản phẩm gia dụng** bạn đã chọn ở Module 1. Nếu bạn đã hoàn thành [Module 2 — Bài tập 4](../module-02-prompt-mastery/bai-tap.md), hãy dùng **sản phẩm và prompt template** bạn đã tạo.

```
📦 SẢN PHẨM CỦA TÔI: _______________
💰 Hoa hồng: ___%
🎯 Đối tượng mục tiêu: _______________
```

## Bước 1: Lên ý tưởng & Storyboard (15 phút)

### 1.1. Brainstorm bằng Gemini

Mở Google Flow → tab **Chat** → hỏi Gemini:

> Tôi muốn tạo 1 video TikTok affiliate 30 giây cho sản phẩm [tên sản phẩm]. Target đối tượng [mô tả đối tượng] tại Việt Nam. Hãy gợi ý: (1) ý tưởng nội dung phù hợp, (2) cấu trúc video gồm mấy phần, (3) kiểu hook nào hiệu quả nhất.

Ghi lại ý tưởng hay nhất: _______________

### 1.2. Viết storyboard

Điền vào storyboard sau:

| Scene # | Phần (H/D/R/C/A) | Thời lượng | Mô tả visual ngắn | Loại asset |
|---------|-------------------|-----------|-------------------|-----------|
| 1 | **H** — Hook | ___s | | Video / Ảnh |
| 2 | **D** — Demo | ___s | | Video / Ảnh |
| 3 | **D** — Demo | ___s | | Video / Ảnh |
| 4 | **R** — Result | ___s | | Video / Ảnh |
| 5 | **A** — CTA | ___s | | Video / Ảnh |
| **Tổng** | | **___s** | | |

💡 **Nhắc:** Mỗi clip Veo 3.1 tối đa 8 giây. Tổng nên ≥ 30 giây. Tham khảo cấu trúc HDRCA trong [bài học, mục 8.1](bai-hoc.md#81-cấu-trúc-hdrca--5-phần-của-video-affiliate-thành-công).

### ✅ Checklist Bước 1

### 2.2. Viết prompt theo SHARP

Với **mỗi scene cần dùng Veo 3.1 hoặc Imagen 4**, viết prompt đầy đủ theo công thức [SHARP](../module-02-prompt-mastery/bai-hoc.md#2-công-thức-sharp--framework-viết-prompt):

**Scene __: ___________ (Veo 3.1 / Imagen 4)**

> **S** (Scene — Bối cảnh): _______________
> **H** (Hero — Sản phẩm): _______________
> **A** (Action — Hành động): _______________
> **R** (Result — Kết quả): _______________
> **P** (Props — Đạo cụ): _______________

**Prompt ghép hoàn chỉnh:**

*(Lặp lại cho mỗi scene)*

## Bước 4: Ghép video bằng Scene Builder (15 phút)

### 4.1. Sắp xếp timeline

Mở Google Flow → tab **Scene** → kéo thả clips:

```
Timeline của tôi:

[Scene __: ___, __s] → [Scene __: ___, __s] → [Scene __: ___, __s] → [Scene __: ___, __s] → [Scene __: ___, __s]

Transition:   [___]         [___]          [___]          [___]
```

### 4.2. Chọn transition

Với mỗi chuyển cảnh, chọn 1 kiểu:
- **Cut** (cắt nhanh) — cho hook → demo
- **Cross dissolve** (hòa chồng) — cho demo → demo
- **Fade** (mờ dần) — cho result → CTA

### ✅ Checklist Bước 4

- [ ] Đã sắp xếp tất cả clips vào timeline
- [ ] Đã chọn transition cho mỗi chuyển cảnh
- [ ] Đã preview toàn bộ video
- [ ] Video chạy mượt, chuyển cảnh tự nhiên

## 📊 Bài tập phân tích: Tự đánh giá video của bạn

Sau khi hoàn thành video, hãy **phân tích** video của mình theo cấu trúc HDRCA:

### Bảng phân tích

| Phần | Có trong video? | Thời lượng | Đánh giá (1-5 ⭐) | Cần cải thiện gì? |
|------|----------------|-----------|-------------------|-------------------|
| **H** — Hook | Có / Không | ___s | | |
| **D** — Demo | Có / Không | ___s | | |
| **R** — Result | Có / Không | ___s | | |
| **C** — Credibility | Có / Không | ___s | | |
| **A** — CTA | Có / Không | ___s | | |

### Câu hỏi phân tích

1. **Hook** của bạn thuộc kiểu nào? (Vấn đề–Giải pháp / Wow moment / Câu hỏi / Thử thách)
   → Trả lời: _______________

2. Nếu bạn phải **cải thiện 1 scene duy nhất**, bạn sẽ chọn scene nào? Tại sao?
   → Trả lời: _______________

3. So sánh video của bạn với cấu trúc HDRCA trong [bài học](bai-hoc.md#81-cấu-trúc-hdrca--5-phần-của-video-affiliate-thành-công) — thiếu phần nào? Bạn sẽ bổ sung ra sao trong video tiếp theo?
   → Trả lời: _______________

4. Bạn đã dùng bao nhiêu credits? Có cách nào **tiết kiệm hơn** cho video sau không?
   → Trả lời: _______________
